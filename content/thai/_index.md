# บทเรียน GroupDocs API และส่วนของโค้ด

ยินดีต้อนรับสู่ศูนย์บทเรียนอย่างเป็นทางการสำหรับ API การจัดการเอกสาร GroupDocs ทรัพยากรครอบคลุมนี้ให้บทเรียนละเอียด ตัวอย่างโค้ด และแนวปฏิบัติที่ดีที่สุดสำหรับการนำผลิตภัณฑ์ GroupDocs ไปใช้ในแอปพลิเคชัน .NET และ Java ของคุณ

{{% alert color="primary" %}}
**ทรัพยากรสำหรับนักพัฒนา:** แต่ละส่วนผลิตภัณฑ์ด้านล่างมีลิงก์ไปยังบทเรียน .NET และ Java พร้อมตัวอย่างโค้ดที่ใช้งานได้จริงและคู่มือการนำไปใช้ทีละขั้นตอน
{{% /alert %}}
